WebShield — Privacy Policy

Last updated: October 28, 2025

WebShield is an extension by Emir Abdullah Alakus. This page explains what data (if any) WebShield collects and how it is used.

Data collection

WebShield stores configuration and preferences locally using Chrome's storage API. By default it does not collect or send personal information to external servers.

Optional server communication

If you enable any optional features that require remote resources (for example, pattern updates or remote analytics), WebShield will request only the minimal data necessary (non-identifying usage metrics) and will clearly request your consent within the extension UI.

Third-party links

WebShield may include links to external sites (e.g. Ko-fi). These links open external pages — WebShield is not responsible for third-party privacy practices.

Contact

If you have questions about this policy, contact: abdullah@yourdomain.com or open an issue at the project's GitHub repository.

Note: This is a concise privacy summary for the Chrome Web Store. If your extension later collects identifiable data, update this page accordingly.
